User Metrics

Welcome to the Wikimedia Foundation's user metrics API homepage.

This API allows you to select a set of users, also known as a cohort (for example, all users who signed up via the Thank You campaign), select a metric to be computed (for example, their 7-day revert rate) and retrieve a value for each of these users. This is what we call a raw response.

You can also compute a single, aggregate response for the entire cohort (like its mean revert rate).
